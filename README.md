## Install
Install with NPM
```
npm install license-plate --save
```

Install with yarn
```
yarn add license-plate --save
```

## How to use
```
import LicensePlate from 'license-plate';

LicensePlate.make('99xxxx'); // returns '99-XX-XX'
```
